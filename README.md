# prediksi-diabetes-ann-jawa-barat
Aplikasi web prediksi diabetes melitus berbasis Artificial Neural Network (ANN) menggunakan Flask dan visualisasi Chart.js dengan dataset Jawa Barat.

🚀 Built with:
- Flask
- Pandas
- Chart.js
- ANN (Artificial Neural Network)

📊 Features:
- Diabetes prediction by region
- Trend visualization
- Interactive UI
